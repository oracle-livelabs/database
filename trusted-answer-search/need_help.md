# Need help?

## Introduction

In this lab, you will learn how to reach out for help if you have any questions when following this LiveLab and using Oracle Trusted Answer Search.

**Estimated time:** 3 minutes.

### Objectives

- Learn about Oracle Trusted Answer Search documentation and helpful resources.

### Prerequisites

- None

## Task 1: Check the user guide

The user documentation is available at [Oracle Trusted Answer Search User's Guide](https://www.oracle.com/database/trusted-answer-search/).

Please refer to it if you have any questions or need additional guidance.

## Task 2: Download Trusted Answer Search

You can find the official installation download page available at [Get Oracle Trusted Answer Search](https://www.oracle.com/database/technologies/trusted-answer-search-downloads.html).
