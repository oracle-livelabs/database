# Workshop Details

Estimated Time: **1 minute**

## Short Description

Build financial intelligence with Oracle Database 26ai by connecting Seer Bank risk operations, signal search, fraud graph analysis, service coverage, prediction, governed answers, and agent actions to database-backed evidence.

## Long Description

This workshop teaches the database foundation behind the Seer Bank Finance LiveStack application. Learners use Database Actions SQL Worksheet to inspect semantic finance views, dashboard metrics, JSON Relational Duality documents, AI Vector Search with MiniLM embeddings, SQL/PGQ graph traversal, Oracle Spatial coverage queries, persisted OML model scoring, governed data-copilot SQL, and auditable PL/SQL agent tools. The workshop is grounded in the current Finance LiveStack schema and data.

## Workshop Outline

1. Introduction
2. Getting Started
3. Finance Data Foundation
4. Risk and Operations Dashboard
5. Transaction and Case Documents with JSON Relational Duality
6. Risk Signal Intelligence with AI Vector Search
7. Financial Crime Network with Property Graph
8. Client Service and SLA Coverage with Oracle Spatial
9. Predictive Risk, Capacity, and Revenue with OML
10. Governed Data Copilot: Trusted Answers
11. AI Operations Agent Console: Trusted Actions
12. Conclusion
13. Final Quiz

## Prerequisites

- LiveLabs reservation with Autonomous Database 26ai.
- SQL Worksheet access as the main workshop schema user, usually `LLUSER`.
- Finance schema loaded from the current Finance LiveStack source bundle.
