# Introduction

Estimated Workshop Time: 120 minutes

## About this Workshop:

This workshop demonstrates how to combine the power of:

- **AI-assisted development**
- **Prompt-led application generation**
- **MySQL HeatWave GenAI**
- **OCI Generative AI services**
- **Modern full-stack architecture**

to rapidly create intelligent, scalable, and AI-enabled applications.

Rather than treating AI as a simple chatbot, this workshop teaches you how to use AI as a **full-stack application engineering assistant** capable of generating:

- Frontend UI layouts
- Backend APIs
- Database schemas
- Seed data
- AI-powered recommendation systems
- Vector search implementations
- Summarization workflows
- Deployment instructions

Using carefully structured prompts, developers can guide AI systems to generate complete application foundations with modern architecture, clean UI design, modular code structure, and native AI integration.

### Objectives
In this Workshop, you will learn how AI can dramatically accelerates the development lifecycle.

You will learn how to:

- Use AI to generate full-stack applications
- Build production-quality starter applications with prompts
- Rapidly prototype AI-enabled business solutions
- Integrate semantic vector search into applications
- Use MySQL HeatWave native AI capabilities directly from your applications
- Enable OCI Generative AI services securely
- Deploy cloud-ready AI applications faster than traditional workflows

The focus is not just on writing prompts — it is about learning how to **architect applications with AI as your development partner**.

## Workshop Learning Path

---

### Lab 1 — Provision the required resources for the workshop

Set up the foundational development environment and establish secure connectivity to MySQL HeatWave.

---

### Lab 2 — Database Setup and AI Access Configuration

Create application databases, provision secure users, configure access policies, and enable HeatWave access to OCI Generative AI services.

You will learn how to prepare an enterprise-ready database environment that supports AI-powered workloads.

---

### Lab 3 — Prompt-Led AI Application Development

Use structured prompts to generate production-quality AI applications using:

- Python Flask
- MySQL HeatWave

- Modern frontend frameworks

This lab demonstrates how AI can generate:

- Complete frontend interfaces
- Backend APIs
- Database models
- AI search workflows
- Vector embedding integration
- Deployment-ready project structures
