# Integrate Agent Factory Custom Agents in APEX and other external Apps

## Introduction

This is a demo only lab. We'll show how you can integrate custom agents created in Agent Factory in other applications. We'll create an APEX App which will call a custom agent to answer user questions. 

**Estimated time:** 10 minutes.

### Objectives

- Understand how to use Agent URL of published agents
- Understand how to use Agent URL in an APEX App

### Prerequisites

* Familiarity with building custom agents in Agent Factory  
* Basic knowledge of APEX apps
