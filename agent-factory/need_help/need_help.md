# Need help?

## Introduction

In this lab, you will learn how to reach out for help if you have any questions when following this LiveLab and using Oracle AI Database Private Agent Factory.

**Estimated time:** 3 minutes.

### Objectives

- Learn about Oracle AI Database Private Agent Factory documentation and helpful resources.

### Prerequisites

- None.

## Task 1: Check the user guide

The user documentation is available at [Oracle AI Database Private Agent Factory User's Guide](https://docs.oracle.com/en/database/oracle/agent-factory/25.3/paias/introduction.html).

Please refer to it if you have any questions or need additional guidance.

## Task 2: Download installation kit

You can find the official installation kit download page available at [Get Oracle AI Database Private Agent Factory kit](https://www.oracle.com/database/technologies/private-agent-factory-downloads.html#).

## Task 3: Deploy in OCI Marketplace

Check out the [Oracle AI Database Private Agent Factory listing](https://cloudmarketplace.oracle.com/marketplace/en_US/listing/201588705) to deploy the application in your own tenancy.
