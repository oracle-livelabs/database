# Workshop Details

## Introduction

This file captures reusable metadata for the Seer Equity Oracle AI Database for Financial Services workshop.

### Objectives

- Summarize the learner outcome.
- Preserve the source-backed lab outline.
- Keep publication notes separate from lab instructions.

Estimated Time: 5 minutes

## Short description

Explore how Seer Equity uses Oracle AI Database 26ai to power finance operations, risk investigation, spatial service coverage, vector search, JSON duality, VPD, and governed AI workflows.

## Long description

This workshop follows the Seer Equity LiveStack from React screens to Express routes and Oracle Database objects. Learners inspect the schema, run SQL-first checks, trace API calls, query JSON duality views, explore risk signals with vector search, investigate graph relationships, test spatial service coverage, review VPD policy behavior, and examine AI-assisted workflows. The workshop uses the supplied input package as the source of truth and includes a final scored quiz.

## Workshop outline

- Introduction: scenario, architecture, source package, app-flow graphic, and learning goals.
- Getting Started: environment, source package, and validation checks.
- Lab1: Orient to the Seer Equity control tower.
- Lab2: Explore the financial services data foundation.
- Lab3: Query dashboard and operational intelligence.
- Lab4: Search regulatory and market signals.
- Lab5: Investigate financial crime networks and service coverage.
- Lab6: Inspect transactions as rows and JSON documents.
- Lab7: Review predictive risk and revenue analytics.
- Lab8: Ask data questions and inspect governed agent actions.
- Lab9: Complete a scored final quiz with eight questions.

## Source package assumptions

- `frontend-source/` defines learner-facing pages and screen flow.
- `database-source/` defines schema, data, SQL, PL/SQL, security, and database features.
- `backend-source/` and `api-map.md` define the Express API layer and route behavior.
- `source-schema/` contains a sanitized copy of schema scripts for workshop reference.
