# Introduction

## About this Workshop

This hands-on workshop introduces JSON Relational Duality Views in Oracle Database 26ai. You will learn how to expose relational data as updatable JSON documents while preserving relational integrity, performance, and consistency.

In about 15 to 25 minutes, you will build a small relational schema, create single-table and multi-table duality views, and work with the data using document-style CRUD operations and SQL-based JSON querying.

Estimated Workshop Time: 15–25 minutes

### Objectives

In this workshop, you will learn how to:
* Create JSON Relational Duality Views
* Work with relational data through updatable JSON documents
* Perform insert, update, delete, and query operations using JSON and SQL
* Query nested JSON data using SQL functions such as JSON_TABLE

### Prerequisites

This workshop assumes you have:
* Basic familiarity with SQL
* An account created on FreeSQL.com to execute the scripts in each step
